# Danish K | Personal Portfolio
**InnovExa Catalyst - Project 1**

A clean, minimalist, and fully responsive personal portfolio built to showcase foundational web development skills. This project was developed as the first milestone in the InnovExa Catalyst industrial training program.

## Features

* **Responsive Layout:** Adapts seamlessly to desktop and mobile screens using modern CSS Flexbox and media queries.
* **Minimalist UI:** A dark-themed, clutter-free interface that prioritizes readability, user experience, and fast performance.
* **Interactive Elements:** Features a dynamic JavaScript greeting that updates based on the user's local time, alongside a simulated contact form.
* **Semantic Structure:** Organized HTML5 layout for improved accessibility, clean code readability, and easy maintainability.

## 💻 Tech Stack

* **HTML5**
* **CSS3** 
* **Vanilla JavaScript**
