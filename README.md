# Simple Portfolio

A clean, responsive portfolio website created as part of the InnovExa Catalyst Program.

## Features

- **Home**: A welcoming section with a dynamic greeting that changes based on the time of day.
- **About**: A brief introduction to the developer and their goals.
- **Portfolio**: A showcase of projects with interactive hover effects.
- **Contact**: A functional form that displays an alert upon submission.

## Getting Started

### Prerequisites

- A modern web browser (Chrome, Firefox, Safari, Edge).
- [Node.js](https://nodejs.org/) and [npm](https://www.npmjs.com/) (optional, for running the local server).

### Installation

No installation is required. You can run this project using any static file server.

### Running Locally

1.  **Using `http-server`** (Recommended):
    ```bash
    npx http-server
    ```
    Then open `http://localhost:8080` in your browser.

2.  **Using Live Server extension**:
    - Right-click `index.html` and select "Open with Live Server".

3.  **Directly from Browser**:
    - Open `index.html` directly in your browser.

## Technologies Used

- **HTML5**: For the structure of the page.
- **CSS3**: For styling and layout.
- **JavaScript (Vanilla)**: For interactivity and form handling.
